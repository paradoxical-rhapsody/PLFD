# PLFD

## Dependencies

* `R > 3.5.0`

* `Rcpp >= 1.0.2`

* `RcppArmadillo >= 0.9.800`


## Installation

```R
install.packages('PLFD')
```
or
```R
remotes::install_git("https://gitee.com/xu-zc/fastMatrix.git")
```
